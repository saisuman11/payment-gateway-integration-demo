\# 💳 Payment Gateway Integration Demo



A simple web application demonstrating \*\*online payment integration using Razorpay\*\* in Test Mode.  

This project shows how to create orders from a backend server and process payments through a secure payment gateway.



---



\## 🚀 Features



\- Razorpay payment gateway integration

\- Order creation using Node.js backend

\- Test mode payment processing

\- Simple frontend payment button

\- Ready for deployment and demonstration



---



\## 🛠️ Technologies Used



\- Node.js

\- Express.js

\- Razorpay API

\- HTML / JavaScript



---

\## ⚙️ Setup Instructions



\### 1️⃣ Clone the repository

git clone https://github.com/yourusername/payment-gateway-integration-demo.git

cd payment-gateway-integration-demo





---



\### 2️⃣ Install dependencies

npm install





---



\### 3️⃣ Add Razorpay Test Keys



Replace in `server.js` and `index.html`:

YOUR\_TEST\_KEY\_ID

YOUR\_TEST\_KEY\_SECRET





Get keys from Razorpay Dashboard → Test Mode → API Keys.



---



\### 4️⃣ Run the server

node server.js





---



\### 5️⃣ Open in browser

http://localhost:3000/
